## 🌟 विशेषताएं

### मल्टी-मार्केट कवरेज
- **भारतीय बाजार**: NSE, BSE — NIFTY 50, SENSEX, F&O, मिडकैप/स्मॉलकैप
- **अमेरिकी बाजार**: S&P 500, NASDAQ, NYSE (Alpaca API के माध्यम से)
- **क्रिप्टो**: 24/7 ट्रेडिंग — Binance, Coinbase, WazirX, CoinDCX
- **फॉरेक्स**: USD/INR, EUR/USD, GBP/USD और अधिक
- **कमोडिटीज**: सोना, चांदी, कच्चा तेल, प्राकृतिक गैस

### 6 ट्रेडिंग रणनीतियाँ (एक साथ चलती हैं)
1. 📈 **ट्रेंड फॉलोइंग** — EMA क्रॉसओवर + सुपरट्रेंड + ADX
2. 🔄 **मीन रिवर्शन** — बोलिंगर बैंड स्क्वीज + RSI डाइवर्जेंस
3. 🚀 **मोमेंटम ब्रेकआउट** — 52-सप्ताह उच्च ब्रेक + वॉल्यूम सर्ज
4. ⚡ **स्कैल्पिंग** — 1-मिनट चार्ट पैटर्न (NSE इंट्राडे)
5. 🕐 **स्विंग ट्रेडिंग** — 3-10 दिन होल्ड
6. 🎯 **ऑप्शंस रणनीति** — NIFTY/BANKNIFTY वीकली एक्सपायरी

### AI/ML इंजन (4-मॉडल एन्सेम्बल)
- **LSTM** (30% वजन) — 60-दिन अनुक्रम → 3-दिन मूल्य भविष्यवाणी
- **XGBoost** (30% वजन) — 50+ फीचर सिग्नल क्लासिफायर
- **PPO RL एजेंट** (25% वजन) — इष्टतम समय के लिए रीइन्फोर्समेंट लर्निंग
- **FinBERT सेंटीमेंट** (15% वजन) — वित्तीय समाचार NLP विश्लेषण

### हेज फंड-ग्रेड रिस्क मैनेजमेंट
- पोजीशन साइजिंग: ATR-आधारित + VIX-समायोजित
- स्टॉप लॉस: प्रारंभिक ATR×2, ट्रेलिंग ब्रेकईवन/ATR×1.5
- आंशिक निकास: 40% 2R पर, 40% 3R पर, 20% ट्रेल
- दैनिक/साप्ताहिक/मासिक हानि सीमा (3%/7%/15%)
- ड्रॉडाउन प्रोटोकॉल (5%→रूढ़िवादी, 10%→पेपर, 15%→रुकावट)

## 🚀 त्वरित शुरुआत

### आवश्यकताएं
- Node.js 20+
- Python 3.12+
- Docker और Docker Compose (वैकल्पिक)

### 1. सेटअप
```bash
cd Trading

# पर्यावरण फ़ाइल कॉपी करें
cp .env.example .env
# .env में अपनी API कुंजियाँ डालें (पेपर ट्रेडिंग के लिए वैकल्पिक)
```

### 2. डेटाबेस सेवाएं शुरू करें
```bash
cd docker
docker compose up -d postgres redis
cd ..
```

### 3. बैकएंड सेटअप
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

### 4. फ्रंटएंड सेटअप
```bash
cd frontend
npm install
npm run dev
```

### 5. डैशबोर्ड खोलें
**http://localhost:5173** पर जाएं 🎉
